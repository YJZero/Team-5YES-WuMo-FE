## Commit Convention

| prefix | description |
| --- | --- |
| feature | 새로운 기능 추가 |
| fix | 버그 수정 |
| rename | 파일 혹은 폴더명 수정, 이동
| remove | 파일 삭제
| design | UI 디자인 변경
| docs | 문서 수정 |
| style | 코드 포맷팅 |
| refactor | 코드 리팩토링 |
| chore | 빌드 업무 수정, 패키지 매니저 수정 |
